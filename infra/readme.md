# Infrastructure
This folder contains infrastructure-related configurations, such as Docker Compose files.

## Files
- **`docker-compose.yml`** - Defines multi-container Docker applications.

## Usage
To spin up the infrastructure, run:
```sh
docker-compose up -d
```
